
# Working with files

Create file with content.

```bash|{type:'file',path:'/tmp/docable_file'}
hello 
docable
```

Check file.

```bash|{type:'command'}
cat /tmp/docable_file
```

Create a directory with path.

```bash|{type:'file',path:'docable_test/docable_file'}
hello 
docable
```

Remove directory.

```bash|{type:'command'}
rm -rf docable_test
```
